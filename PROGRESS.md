# DragonGlareAlpha 進捗メモ

更新日: 2026-05-11

## 現状サマリー

- `DragonGlareAlpha` は `DragonGlareAlpha.csproj` を基準に管理する MonoGame / Windows 向けRPG風プロトタイプです。
- 現在のアプリバージョンは `1.4.5` です。
- 画面遷移は「モード選択 → 言語選択 → 名前入力 → フィールド → バトル / ショップ / 銀行 / ダンジョン」まで実装済みです。
- 武器、むねあて、あたま、こて、レギンス、ブーツの装備スロットを保持し、戦闘計算とセーブデータに反映します。

## 直近の主な更新

### Alpha 1.4.2

- 戦闘結果のメッセージを一括表示から自動ステップ送りに変更し、各行の表示に合わせて演出を再生するようにした。
- 勝利報酬を「経験値/G → レベルアップ → ドロップ」の順で個別表示するように整理。
- 攻撃、防御、敵攻撃、アイテム使用、撃破などの戦闘演出を `BattleVisualCue` ごとに分離し、各メッセージに合わせて個別再生するようにした。
- `AntiCheatService` の起動時プロセス検出を同期実行にし、実行中の非同期スキャンをスレッドセーフ化してスキャン間隔を制御するようにした。
- 戦闘演出キューとチート検出周辺のテストを追加・更新。

### Alpha 1.4.0

- フィールド遷移とイベント定義を `FieldContent` に分離。
- フィールドからダンジョンへ移動できる入口と、ダンジョンから戻る遷移を追加。
- フィールド上の看板と宝箱イベントを追加し、宝箱報酬と完了済みイベントの保存に対応。
- `SpriteFontPlus` 依存を外し、TTF フォントを直接描画する方式に変更。
- フィールドイベント、マップ生成、マップ遷移、セーブデータ移行のテストを追加・更新。
- フィールド移動の入力遅延を抑えるため、移動アニメーションを短縮し、通常の進捗保存を入力更新フレームから遅延実行するよう変更。
- 名前入力画面の日本語テーブルに小文字かなを追加し、行数増加に合わせて配置を調整。

### ショップUI

- ショップの最初の選択肢で、文字とウィンドウ枠が重ならないように選択肢の位置とヘルプ枠の高さを調整。
- 選択矢印の縦位置を文字行に合わせて修正。
- 装備情報欄に `ぶき`、`あたま`、`むねあて`、`こて`、`レギンス`、`ブーツ` を個別表示するよう変更。
- 装備スロット表示のラベルを、ユーザーが確認しやすい部位名に整理。

### フィールドUI

- フィールド右側のステータス欄で、装備数だけでなく現在装備している武器、防具各部位の名前を表示するよう変更。
- 表示対象は `ぶき`、`あたま`、`むね`、`こて`、`レギンス`、`ブーツ`。

### バージョン管理

- ゲームパッド対応に合わせて `AppVersion` を `1.4.0` に更新。
- メインプロジェクトを `DragonGlareAlpha.csproj` に統一し、未参照の旧プロジェクト定義を削除。
- アプリ起動時に MuiCache の `ApplicationCompany` と `FriendlyAppName` を現在のバージョン表示へ同期するよう変更。
- 今後、変更内容に応じて `.csproj` のバージョンを更新します。

### ゲームパッド対応

- 十字キー / 左スティックを移動・メニュー選択に対応。
- A / Start を決定、B / Back をキャンセル、X をステータス / 削除、Y を決定系の補助入力に割り当て。
- LB をバトル、RB をショップのショートカットとして追加。
- ゲーム内の操作説明と看板テキストをゲームパッド表記に更新。

## 実装済み機能
