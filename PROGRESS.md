# DragonGlareAlpha 進捗メモ

更新日: 2026-04-13

## 現状サマリー

- `DragonGlareAlpha` は `net10.0-windows` を対象にした WinForms 製のRPG風プロトタイプです。
- 画面遷移は「モード選択 → 言語選択 → 名前入力 → フィールド → バトル / ショップ」まで実装済みです。
- 2026-04-13 現在、バージョン `v1.3.0` に到達し、戦闘UI強化、会話ポートレート追加、全画面切替を実装しました。

## 今回の主な更新

### システム・UI整理
- プロジェクトの製品バージョンを `1.3.0` に更新。
- ウィンドウ枠を立体感のある二重線フレームへ刷新。
- 戦闘画面下部を「敵情報・コマンド・メッセージ一体型」レイアウトへ再構成。
- `F11` でフルスクリーンとウィンドウ表示を即時切り替えられるよう改善。
### 未完了タスク
- **演出の細分化**: 戦闘時のメッセージ送りやキャラクターモーションの改善。
## 🛠️ 新規実装：銀行（ぎんこう）システム
ショップ機能を応用し、以下の3つのコマンドを持つ銀行機能を新設する。
1. **あずける**: 所持金を預け、全滅時のゴールド失踪リスクを回避する。
2. **ひきだす**: 必要な時に預金からゴールドを取り出す。
3. **かりる**: 
   - 借金をして強力な装備を早期に購入可能にする。
   - 利息（歩数や戦闘による加算）や、返済不能時のペナルティ（装備没収等）の検討。
## 残タスク候補
- 細部の改善

### バトル
- 戦闘コマンドに `ぼうぎょ` と `そうび` を追加。
- `どうぐ` をサブメニュー化し、戦闘中に実際に使えるよう修正。
- `そうび` では所持武器・防具の付け替え、`ぼうぎょ` では被ダメージ軽減を実装。

### アセット
- 会話ウィンドウに NPC portrait を追加し、案内役・町の子ども・城の兵士・みはりに新規イラストを割り当て。
- portrait 用アセット配置先として `Assets/Portraits/NPC` を追加。
